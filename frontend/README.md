# GolfMinsk Live Scoring — Frontend

React + TypeScript + Vite фронтенд для системы лайв-скоринга Golf Club Minsk.

## Разработка

```bash
npm install
npm run dev       # запускает на :8080, проксирует /api → :3001
```

## Стек

- React 18 + TypeScript
- Vite + shadcn/ui + Tailwind CSS
- Zustand (локальное хранение раундов)
- React Router v6
